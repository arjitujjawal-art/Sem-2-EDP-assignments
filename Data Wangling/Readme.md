## 📖 Theory

### 🔹 What is Data Preprocessing? 🤔
**Data preprocessing** is the process of cleaning and transforming raw data into a structured and usable format. It is an essential step before performing data analysis or applying machine learning algorithms.

---

### 🔹 Need for Data Preprocessing 🚀
Raw data may contain:
- Missing values
- Duplicate records
- Inconsistent formats
- Errors or noise

Preprocessing helps to improve data quality and ensures accurate results.

---

### 🔹 Missing Values ❓
Missing values occur when no data is stored for a variable in a dataset. These can affect analysis and model performance if not handled properly.

---

### 🔹 Techniques to Handle Missing Values 🧠

#### 1️⃣ Removing Missing Data
- Delete rows or columns with missing values
- Used when missing data is very small

#### 2️⃣ Filling Missing Values (Imputation)
- Replace missing values with:
- Mean
- Median
- Mode
- Maintains dataset size

#### 3️⃣ Forward Fill and Backward Fill
- Fill missing values using previous or next data values
- Useful for time-series data

---

### 🔹 Other Preprocessing Steps 📚

#### 📌 Data Cleaning
- Removing duplicates
- Correcting errors
- Standardizing formats

#### 📌 Data Transformation
- Normalization
- Scaling
- Encoding categorical data

#### 📌 Data Reduction
- Reducing dataset size while keeping important information

- Ensures accurate analysis
- Reduces errors in results
