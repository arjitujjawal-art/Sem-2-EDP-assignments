### 3. Data Types & Mutability
Python is **dynamically typed**, meaning the interpreter automatically detects the data type at runtime. 
* **Mutable Types:** Can be changed after creation (e.g., Lists, Dictionaries).
* **Immutable Types:** Cannot be modified once defined (e.g., Strings, Tuples).
* **Constants:** Variables intended to remain unchanged throughout the program.

### 4. User Interaction
* **Input:** Handled via the `input()` function. Since `input()` defaults to a string, **Type Casting** (e.g., `int()` or `float()`) is necessary for numerical operations.
* **Output:** Handled via the `print()` function to display results to the console.

## ðŸ”¢ Operators in Python
I experimented with five primary categories of operators:

1. **Arithmetic:** Basic math (`+`, `-`, `*`, `/`).
2. **Assignment:** Giving values to variables (`=`, `+=`, `-=`).
3. **Comparison:** Evaluating relationships between values (`==`, `!=`, `>`, `<`).
4. **Logical:** Combining conditional statements (`and`, `or`, `not`).
5. **Bitwise:** Performing operations at the binary level (similar to C).
