### 3. Jupyter Notebook vs. Google Colab
| Feature | Jupyter Notebook (Local) | Google Colab (Cloud) |
| :--- | :--- | :--- |

| **Internet** | Works offline | Requires active connection |

## üõ†Ô∏è Installation & Setup

### Local Setup (Anaconda)
1.  **Download:** Grab the installer for your OS from [Anaconda.com](https://www.anaconda.com/).
2.  **Installation:** Run the setup wizard. *Tip: Ensure you check the box to initialize Anaconda properly.*
3.  **Launch:** Open the **Anaconda Navigator** and launch **Jupyter Notebook**.

### Cloud Setup (Google Colab)
1.  Navigate to [colab.research.google.com](https://colab.research.google.com).
2.  Sign in with a Google Account.
3.  Click **New Notebook** to start coding immediately.
